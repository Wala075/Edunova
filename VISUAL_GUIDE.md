## 🔄 Transitions et Animations

### Checkbox Coché
```
État: Non coché
↓ (Utilisateur clique)
État: Coché
↓ (Instantané)
Question apparaît (visible = true)
TextField reçoit le focus
```

### Vérification Réussie
```
État: Question visible
↓ (Utilisateur clique "Vérifier")
Validation de la réponse
↓ (Si correct)
Message ✅ "Vérification réussie"
Question disparaît (visible = false)
```

### Vérification Échouée
```
État: Question visible
↓ (Utilisateur clique "Vérifier")
Validation de la réponse
↓ (Si incorrect)
Message ⚠ "Réponse incorrecte"
TextField se vide
Nouvelle question générée
Focus revient au TextField
```

## ♿ Accessibilité

### Keyboard Navigation
- Tab: Naviguer entre les champs
- Space: Cocher/décocher le checkbox
- Enter: Cliquer sur le bouton

### Screen Reader
- Checkbox: "Je ne suis pas un robot, checkbox"
- Question: "Combien font 5 + 3 ?"
- TextField: "Entrez votre réponse, text input"
- Bouton: "Vérifier, button"

### Contraste
- Texte sur fond: Ratio 4.5:1 (WCAG AA)
- Bordures: Visibles et distinctes
